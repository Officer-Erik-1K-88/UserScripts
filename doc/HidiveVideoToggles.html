Hidive Video Toggles

Go Back to Home Page

Currently, we only have one toggle. And one special.

Toggles:

  • Subtitles

    Toggle with: C key

    Description:

    Switch between Off and last selected. If there is no value in last selected, then it will default to English.

    The selected option will be saved to a value.

Special:

  • Scrollbar Fix

    Hide the scrollbar on HIDIVE video pages but keep scrolling functionality. This is a fix for the browsers that, for some reason, have a scrollbar appear on the Hidive video player page.
